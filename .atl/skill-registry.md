### agents.md — Compact Rules

**Sub-agent domains (strict ownership):**
- `api-agent` → `src/lib/dropea/**` — GraphQL client, queries, mutations, types
- `catalog-agent` → `src/app/[locale]/productos/**`, `src/components/product/**`
- `cart-agent` → `src/app/[locale]/carrito/**`, `src/app/[locale]/checkout/**`, `src/hooks/useCart.ts`
- `webhook-agent` → `src/app/api/webhooks/dropea/route.ts`
- `i18n-agent` → `src/messages/**`, `src/middleware.ts`
- `seo-agent` → `generateMetadata` en páginas, `sitemap.ts`, `robots.ts`
- `ui-agent` → `src/components/ui/**`, `src/components/layout/**`, `tailwind.config.ts`

**Stack rules:**
- Next.js 15 App Router — Server Components por defecto, `"use client"` solo si necesario
- `graphql-request` para fetching (NO Apollo Client)
- Mutations vía Server Actions (`"use server"`)
- ISR para páginas de producto: `export const revalidate = 3600`
- Zustand para estado del carrito con persistencia en localStorage

**Dropea API rules:**
- Endpoint: `https://api.dropea.com/graphql/dropshippers`
- Auth: header `x-api-key: ${DROPEA_API_KEY}`
- Scopes: Producto (Listar/Ver), Orden (Listar/Ver/Crear/Confirmar/Cancelar/Actualizar), Tienda (Listar/Ver), Incidencia (Listar/Ver/Resolver), Usuario (Ver)
- Schema completo via introspección — no asumir campos

**Shipping rules (Dropea):**
- Servicio 24 por defecto (más barato): 0-1kg €3.68, 1-2kg €3.94, 2-3kg €4.20, 3-5kg €4.73, 5-10kg €6.20, extra/kg €0.37
- Calcular desde peso del producto; fallback tarifa plana €3.68 si no hay peso
- Config via env: `NEXT_PUBLIC_SHIPPING_COST` y `NEXT_PUBLIC_FREE_SHIPPING_THRESHOLD`

**Mercado ES/PT rules:**
- IVA España 21%, Portugal 23% — mostrar precio con IVA incluido
- Páginas legales obligatorias: Aviso Legal, Privacidad, Devoluciones (14 días), Cookies + banner RGPD
- Pagos: SumUp (primario), PayPal, Contra reembolso (Dropea nativo)
